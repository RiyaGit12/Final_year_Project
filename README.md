#  AI Based Analysis of Social Media Data for Public Health Domain

A Final Year B.Tech Project by **Riya Kumari** (Roll: 22-CS-01)  
Government Engineering College, Lakhisarai (2022–2026)

##  About This Project
This project analyzes real-time health news using Artificial Intelligence 
and Machine Learning to monitor public health trends and detect disease outbreaks.

##  Features
- ✅ Real-time health news collection using NewsAPI
- ✅ Text preprocessing using NLP techniques
- ✅ Sentiment analysis (Positive, Negative, Neutral)
- ✅ 3 ML Models: Naive Bayes, Decision Tree, SVM (84% accuracy)
- ✅ Real-time Disease Outbreak Detector by country
- ✅ Beautiful Web App UI using Streamlit

##  Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLTK
- Pandas, NumPy
- Matplotlib, Seaborn
- NewsAPI

##  How to Run
1. Clone the repository
   git clone https://github.com/RiyaGit12/Final_year_Project.git

2. Install dependencies
   pip install -r requirements.txt

3. Run the app
   streamlit run app.py
